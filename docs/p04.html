Spotify Data Analysis

In this portfolio, I decided to analyze my spotify data to learn more about my musical preferences.

Loading in my spotify data

json_files <- list.files(path = "p04/Spotify Account Data", 
                         pattern = "StreamingHistory_music.*\\.json",
                         full.names = TRUE)

raw <- json_files %>%
  map(fromJSON) %>%
  bind_rows()

# Preview the raw data
glimpse(raw)

Cleaning the Data

df <- raw %>%
  rename(
    timestamp = endTime,
    ms_played = msPlayed,
    track     = trackName,
    artist    = artistName
  ) %>%
  mutate(
    timestamp   = ymd_hm(timestamp),  # note: ymd_hm not ymd_hms, no seconds in basic format
    date        = as_date(timestamp),
    hour        = hour(timestamp),
    day_of_week = wday(timestamp, label = TRUE, abbr = FALSE),
    month       = floor_date(date, "month"),
    minutes_played = ms_played / 60000
  ) %>%
  filter(
    !is.na(track),
    ms_played > 30000
  )

Top Artists

df %>%
  count(artist, sort = TRUE) %>%
  slice_head(n = 15) %>%
  mutate(artist = fct_reorder(artist, n)) %>%
  ggplot(aes(x = n, y = artist)) +
  geom_col(fill = "plum4") +
  labs(
    title = "My Top 15 Most Played Artists",
    x = "Number of Plays",
    y = NULL
  ) +
  theme_minimal(base_size = 13)

Top Tracks

Notes

Data covers the past year of streaming history from Spotify’s data export. Plays under 30 seconds were excluded as likely skips and podcasts were excluded (no track metadata).
