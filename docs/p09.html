Overview

This report presents the analyses for the Time Task of the Social Engagement Reinforcement Learning paradigm. Participants provided ratings of how different amounts of seconds spent with someone else would motivate them to participate in a Jenga activity (+15 secs, 0 secs, -15 secs). They also rated how much time, if given the choice, they would like to spend playing with another student. Afterwards, they completed a reinforcement learning task where outcomes varied in how much time they spent with other people (same as explicit ratings: positive = + 15 secs, neutral = 0 seconds, negative = - 15 seconds).

The measure of interest is the computationally inferred Reward Value (cRV / k parameter), which captures the reward values each time outcome through their behavior.

dv_vars_T <- c("PosvsNegTime_diff_imv", "PosvsNegTime_diff_explicit")

cor_table_fn <- function(df, dv_vars, trait_vars) {
  results <- list()
  for (dv in dv_vars) {
    dv_results <- lapply(trait_vars, function(trait) {
      res <- cor.test(df[[dv]], df[[trait]], use = "complete.obs")
      data.frame(
        DV       = dv,
        Trait    = trait,
        r        = round(as.numeric(res$estimate), 3),
        p        = round(res$p.value, 3),
        CI_lower = round(res$conf.int[1], 3),
        CI_upper = round(res$conf.int[2], 3)
      )
    })
    results[[dv]] <- do.call(rbind, dv_results)
  }
  do.call(rbind, results)
}
